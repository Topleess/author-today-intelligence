# Roadmap

## Available now

- documented API collection;
- normalized JSON snapshots;
- SQLite storage and latest-state report;
- agent workflow and evidence rules;
- offline tests.

## Next useful milestone

A 14-day pilot with 6-hour captures, followed by an evidence report: fastest-growing works, ranking movement, chapter/update timing, data-quality failures, and explicit unknowns.

## After validation

1. Events table and delta reports.
2. Work details and chapter connector.
3. Optional comment intelligence with retention/deletion controls.
4. Yandex Metrica import for commercial-author funnels.
5. Read-only web interface focused on decisions.
6. Contextual agent that cites stored evidence.

## Not validated yet

- willingness to pay;
- exact paid-sale attribution;
- reliable ranking causality;
- production stability of undocumented endpoints;
- which interface decisions an author uses repeatedly.
